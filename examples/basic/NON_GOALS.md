# NON_GOALS.md - Task Manager API

## Out of Scope

### Features
- Real-time updates (WebSocket)
- Task sharing/collaboration
- Email notifications
- Task categories or tags
- Bulk operations

### Integrations
- Third-party OAuth (Google, GitHub)
- External calendar sync
- File attachments

### Platforms
- Mobile app
- CLI tool
- Browser extension

## Never Include

1. Never store passwords in plaintext
2. Never skip authentication on protected routes
3. Never expose sensitive data in API responses
